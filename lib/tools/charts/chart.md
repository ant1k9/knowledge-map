*Labels*: #Stats #Charts #Go

*Examples*:

```bash
$ cat testdata/bar.csv | chart bar
$ cat testdata/line.csv | chart line
$ cat testdata/pie.csv | chart
$ history | awk '{print $1}' | chart
```
