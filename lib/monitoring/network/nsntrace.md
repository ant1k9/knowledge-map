[nsntrace](https://github.com/nsntrace/nsntrace)

*Description*: Perform network trace of a single process by using network namespaces.

*Labels*: #C #Sniffing #tcpdump

*Links*:
  - https://sysadmin.pm/nsntrace/

*Examples*:

```bash
$ nsntrace --use-public-dns wget -q -O /dev/null https://sysadmin.pm/
```
